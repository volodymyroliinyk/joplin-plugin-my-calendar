# Recurring Weekly Event

This event repeats every Tuesday and Thursday.

```mycalendar-event
title: Gym Session
start: 2025-01-07 08:00
end: 2025-01-07 09:30
repeat: weekly
repeat_interval: 1
byweekday: TU,TH
color: #2ecc71
```

The calendar will automatically calculate and display occurrences for every Tuesday and Thursday.
